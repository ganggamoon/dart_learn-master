Control Flow & Looping

"Control Flow adalah cara program mengatur alur eksekusi kode."

1. Percabangan (Conditional Statement)

Percabangan digunakan untuk menjalankan blok kode berdasarkan kondisi (true atau false).

A. If, Else If, dan Else

if → Mengecek kondisi utama.

else if → Mengecek kondisi tambahan jika sebelumnya salah.

else → Dijalankan jika semua kondisi salah.

Cara Kerja:

  1. Membaca kondisi dari atas ke bawah.
  2. Jika true, blok dijalankan.
  3. Pengecekan berhenti setelah kondisi benar ditemukan.
[Image of if-else-if flow chart]
cek_suhu.dart 
void main() {
  double suhu = 38.5;

  if (suhu >= 38) {
    print("Kamu Demam");
  } else if (suhu >= 37) {
    print("Suhu Normal Tinggi");
  } else {
    print("Kamu Sehat");
  }
}

B. Switch Case

Switch digunakan untuk memilih aksi berdasarkan nilai yang sudah pasti secara langsung (bukan logika perbandingan besar/kecil).

switch

Variabel yang diperiksa

case

Nilai pembanding

break

Hentikan proses

default

Pilihan terakhir

cek_grade.dart
void main() {
  String grade = 'B';

  switch (grade) {
    case 'A':
      print("Istimewa");
      break;
    case 'B':
      print("Bagus");
      break;
    default:
      print("Tetap Semangat");
  }
}

2. Perulangan (Looping)

Digunakan untuk menjalankan kode yang sama secara berulang-ulang sampai kondisi berhenti terpenuhi.

A. For Loop

Digunakan jika jumlah perulangan sudah diketahui di awal.

[Image of for loop flow chart]
for_example.dart
void main() {
  for (int i = 1; i <= 5; i++) {
    print("Angka $i");
  }
}

B. While Loop

Digunakan jika jumlah perulangan belum pasti dan bergantung kondisi.

while (baterai > 0) {
  print("Nyala");
  baterai--;
}

C. Do-While Loop

Minimal dijalankan satu kali sebelum mengecek kondisi.

do {
  print("Angka: $a");
  a++;
} while (a <= 3);

D. For-In Loop

Khusus digunakan untuk mengakses elemen dalam List atau koleksi data.

belanja.dart
var belanjaan = ['Beras', 'Telur'];

for (var item in belanjaan) {
  print("Beli: $item");
}

3. Kontrol Perulangan

BREAK

Menghentikan perulangan secara paksa saat kondisi tercapai.

if (i == 4) break;

CONTINUE

Melewati sisa kode di iterasi ini dan lanjut ke putaran berikutnya.

if (i == 2) continue;

Ringkasan Materi

  • If-Else: Kondisi logika kompleks
  • Switch: Pilihan nilai tetap (Menu)
  • For: Jumlah putaran diketahui
  • While: Bergantung pada kondisi
  • Do-While: Minimal jalan 1 kali
  • Break: Stop Paksa
