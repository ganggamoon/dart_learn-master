# Dart with Sherly — Learning Dart by Examples

This repository contains a set of small Dart examples and lesson notes used for teaching and self-study. Each `Pertemuan` (meeting/lesson) folder contains example `.dart` files, notes, and small demos that illustrate core Dart concepts.

**Contents**

- `Pertemuan 1/` — Setup, basic syntax, types, null-safety, and introductory examples.
- `Pertemuan 2/` — Arithmetic, operators, comparison, and logical operators.
- `Pertemuan 3/` — Control flow: `if`/`else`, `switch`, and loops.
- `Pertemuan 4/` — Functions and related examples.
- `Pertemuan 5/` — Collections: lists and related exercises.

**Quick start — prerequisites**

- Install the Dart SDK (stable). See https://dart.dev/get-dart for install instructions.

**Running examples**

You can run individual example files with the Dart CLI. From the repository root run:

```bash
dart run Pertemuan\ 1/hello_dart.dart
```

Or run any example directly, for example:

```bash
dart run Pertemuan\ 3/looping-for.dart
```

Note: Windows paths with spaces may require quoting in PowerShell or CMD. Replace `\` and spaces appropriately for your shell.
