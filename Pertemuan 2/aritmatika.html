Materi: Operator Aritmatika

Operator Aritmatika

Operator aritmatika digunakan untuk melakukan operasi matematika dasar pada tipe data numerik seperti int dan double.

Operator Fungsi Contoh Hasil
+Penjumlahan5 + 38
-Pengurangan5 - 32
*Perkalian5 * 315
/Pembagian5 / 22.5
~/Pembagian bulat5 ~/ 22
%Sisa bagi5 % 21
++Incrementa++a+1
--Decrementa--a-1

Operator Penjumlahan (+)

Operator + digunakan untuk menjumlahkan dua nilai.

penjumlahan.dart 
void main() {
  var a = 5;
  var b = 3;
  var c = a + b;
  print(c); // 8
}

Operator Pengurangan (-)

Digunakan untuk mengurangi satu nilai dengan nilai lainnya.

pengurangan.dart
void main() {
  var a = 10;
  var b = 4;
  var c = a - b;
  print(c); // 6
}

Perkalian (*)

Digunakan untuk mengalikan satu nilai dengan nilai lainnya.

perkalian.dart 
void main() {
  var a = 10;
  var b = 4;
  var c = a * b;
  print(c); // 40
}

Pembagian Desimal (/)

Digunakan untuk membagi dua nilai dan hasilnya selalu bertipe double.

pembagian.dart 
void main() {
  var a = 10;
  var b = 4;
  var c = a / b;
  print(c); // 2.5
}

Pembagian Bulat (~/)

Digunakan untuk membagi dan mengambil angka bulatnya saja.

pembagian_bulat.dart 
void main() {
  var a = 10;
  var b = 4;
  var c = a ~/ b;
  print(c); // 2
}

Sisa Bagi (%)

Digunakan untuk mengambil sisa dari hasil pembagian (Modulus).

modulus.dart 
void main() {
  var a = 10;
  var b = 4;
  var c = a % b;
  print(c); // 2
}

Increment (++)

Digunakan untuk menambah 1 pada nilai variabel.

increment.dart 
void main() {
  var a = 10;
  a++; 
  print(a); // 11
}

Decrement (--)

Digunakan untuk mengurangi 1 pada nilai variabel.

decrement.dart 
void main() {
  var a = 10;
  a--; 
  print(a); // 9
}
