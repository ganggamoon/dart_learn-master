Operator Perbandingan

Digunakan untuk membandingkan dua nilai. Hasil akhirnya selalu berupa nilai boolean true atau false.

Operator Fungsi Contoh
==Sama dengan5 == 5 (true)
!=Tidak sama dengan5 != 3 (true)
>Lebih besar10 > 5 (true)
<Lebih kecil3 < 8 (true)
>=Lebih besar atau sama5 >= 5 (true)
<=Lebih kecil atau sama4 <= 10 (true)

Sama Dengan (==)

equality.dart 
void main() {
  print(10 == 10); // true
  print("Apel" == "Mangga"); // false
}

Tidak Sama Dengan (!=)

inequality.dart 
void main() {
  print(5 ! = 10); // true
}

Lebih Besar / Kecil

relational.dart 
void main() {
  var kkm = 75;
  var nilai = 80;

  print(nilai > kkm);  // true
  print(nilai <= kkm); // false
}
