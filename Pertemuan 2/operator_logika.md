# Operator Logika

Operator logika digunakan untuk menggabungkan atau membalik kondisi yang menghasilkan nilai boolean, yaitu true atau false. Operator ini biasanya digunakan bersama operator perbandingan untuk membuat keputusan dalam program.

## Operator AND (&&)

Operator && digunakan untuk mengecek dua atau lebih kondisi.
Hasilnya akan bernilai true hanya jika semua kondisi bernilai true.

Contoh penggunaan:
- Digunakan saat **semua** syarat harus terpenuhi
- Misalnya: nilai lulus **dan** kehadiran mencukupi

## Operator OR (||)

Operator || digunakan untuk mengecek beberapa kondisi.
Hasilnya akan bernilai true jika salah satu kondisi bernilai true.

Contoh penggunaan:
- Digunakan saat **cukup salah satu** syarat terpenuhi
- Misalnya: lulus ujian **atau** ikut remedial

## Operator NOT (!)

Operator ! digunakan untuk membalik nilai boolean.
- true menjadi false
- false menjadi true

Contoh penggunaan:
- Mengecek kondisi kebalikan
- Misalnya: pengguna belum login
