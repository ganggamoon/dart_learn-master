Materi: Operator Logika

Operator Logika

Operator logika digunakan untuk menggabungkan atau membalik kondisi yang menghasilkan nilai boolean.

&&

Operator AND

Hasilnya true hanya jika semua kondisi bernilai true.

logika_and.dart 
void main() {
  bool nilaiLulus = true;
  bool absenBagus = true;

  // Syarat: Lulus DAN Absen Bagus
  print(nilaiLulus && absenBagus); // true
}
||

Operator OR

Hasilnya true jika salah satu saja kondisi bernilai true.

logika_or.dart 
void main() {
  bool lulusUjian = false;
  bool ikutRemedial = true;

  // Syarat: Lulus Ujian ATAU ikut Remedial
  print(lulusUjian || ikutRemedial); // true
}
!

Operator NOT

Digunakan untuk membalik nilai boolean.

logika_not.dart 
void main() {
  bool sedangLogin = false;

  print(!sedangLogin); // true
}
